Latihan 2 — Navbar Responsif

Pada desktop, item navigasi berjajar horizontal (flex-direction: row, nilai default Flexbox).

Saat layar diperkecil hingga ≤ 768px, media query mengubah flex-direction menjadi column sehingga item nav tersusun secara vertikal, lebih sesuai untuk tampilan mobile.
