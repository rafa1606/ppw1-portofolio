Latihan 3 — Kombinasi min-width dan max-width

Rule @media (min-width: 600px) and (max-width: 1024px) hanya aktif saat lebar layar di antara 600px dan 1024px (ukuran tablet). Pada mobile (< 600px) dan desktop (> 1024px), rule ini tidak berlaku.

Apa yang terjadi pada .container?
Di desktop (> 1024px) dan mobile (< 600px), elemen .container menggunakan flex-direction: row (default) sehingga box-box berjajar horizontal. Saat layar berada di rentang 600px–1024px (tablet), media query mengubahnya menjadi flex-direction: column sehingga box-box tersusun vertikal.
